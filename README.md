# Habitly - Habit Tracker

## Overview
Habitly is a beautiful habit tracking app that helps users build positive habits through streaks, reminders, and visual progress tracking.

## Features

### Habit Management
- **Create Habits**: Add custom habits
- **Track Progress**: Daily check-ins
- **Streaks**: Maintain daily streaks
- **Categories**: Organize habits

### Motivation
- **Visual Progress**: Charts and rings
- **Celebrations**: Confetti on completion
- **Reminders**: Daily notifications
- **Statistics**: Completion rates

## Tech Stack

```yaml
Framework: Flutter 3.x
Language: Dart
State Management: Riverpod
Database: Hive
Charts: fl_chart
Animations: Lottie, Confetti
Notifications: flutter_local_notifications
Haptics: vibration
```
